# Healthy Pro Roadmap

## Next Candidates

1. 动作详情页
   - 器械图、动作步骤、座椅/握把调整、常见错误、替代动作。
   - MVP 先用图片和结构化文字，不做视频或 GIF。

2. 训练中辅助
   - 当前动作大卡片、完成组数、组间休息倒计时、下一组/下一动作。
   - 目标是让训练页像陪练，而不只是训练后记录器。

3. 动作进步
   - 融入「我的」页，展示有记录的力量动作进步。
   - 先看重量、次数和主观感觉趋势；跑步机等心肺动作后续单独做。

4. 计划版本历史
   - 当前只保留上一版和 AI 原始计划。
   - 后续可展示完整版本列表、调整原因和差异。

## Deferred

- AI 实时问答：等模型 API key 和安全边界成熟后再做。
- 视频/GIF 动作教学：先不做，加载和维护成本高；未来优先短 MP4 或外部视频服务 URL。
- 全站公开排行：不做。好友排行仅限互相确认的好友，默认不公开。
