# Healthy Pro WeApp

Healthy Pro 的微信小程序端。当前是可预览的 MVP：可以导入微信开发者工具，完成基础评估、生成训练计划、按记录重新调整计划、编辑计划、恢复旧版计划、保存训练记录；在微信云开发可用时，会按当前微信 `openid` 同步数据，云端失败时保留本地可用。

## 导入方式

1. 打开微信开发者工具。
2. 关闭当前旧项目，或从菜单选择“项目 / 导入项目”。
3. 项目目录选择：

   `/Users/bytedance/healthy-pro/healthy-pro-weapp`

4. AppID：

   当前项目已配置正式小程序 AppID。若只是本地看页面，可以用开发者工具预览；要验证微信身份、云数据库和体验版上传，必须使用正式 AppID 和对应云开发环境。

5. 项目名称可以填：

   `Healthy Pro WeApp`
