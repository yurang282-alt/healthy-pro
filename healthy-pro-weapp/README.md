# Healthy Pro WeApp

Healthy Pro 的微信小程序端。当前是第一阶段 mock 版：可以导入微信开发者工具预览页面、完成基础评估、生成训练计划、按记录重新调整计划、编辑计划、恢复旧版计划、保存本地训练记录。

## 导入方式

1. 打开微信开发者工具。
2. 关闭当前旧项目，或从菜单选择“项目 / 导入项目”。
3. 项目目录选择：

   `/Users/bytedance/healthy-pro/healthy-pro-weapp`

4. AppID：

   当前 mock 预览可以先用开发者工具默认的 `touristappid`。后续要接微信登录、云数据库和体验版上传时，再替换为正式小程序 AppID。

5. 项目名称可以填：

   `Healthy Pro WeApp`

## 下一步需要的东西

拿到云开发环境 ID 后，继续接入：

1. `wx.login`
2. 云函数获取用户 `openid`
3. 云数据库保存评估、计划、训练记录、反馈和好友数据
